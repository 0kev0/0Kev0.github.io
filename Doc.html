Buttons types

Contamos con diversos tipos de botones que en una serie de colores pensado para que sean simples y eficases.
la clase button_round es el boton defecto de la libreria, el tiene sus diferentes colores deacuerdo a la paleta de colores utilizada.

por ejemplo, el boton base pertenese a la clase

class="buttontest"


Buttons Utylities

Las utilidades relacionada con los botones comiensan con la palabra

set_

por ejemplo para aumentar el tamano de un boton se puede utilizar

set_btn_grande
